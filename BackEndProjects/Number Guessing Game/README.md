# Number Guessing Game

## Overview

This is a simple number guessing game where the player has to guess a randomly generated number within a certain range. The game provides feedback on whether the guess is too high, too low, or correct.

## Rules

1. The game will randomly select a number between a specified range (e.g., 1 to 100).
2. The player will have to input their guess.
3. After each guess, the game will inform the player if their guess is too high, too low, or correct.
4. The game continues until the player guesses the correct number.
5. Players can choose to play again after winning.

## Installation Instructions

1. Navigate to the project directory:
   ```bash
   cd Number Guessing Game
   ```
2. Install the required packages:
   ```bash
   pnpm install
   ```

## Usage

To start the game, run the following command:

```bash
pnpm start
```
