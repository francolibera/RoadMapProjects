# Roadmap Projects

Collection of projects completed from [roadmap.sh](https://roadmap.sh/dashboard). This repository acts as a tidy index to navigate exercises and track progress.

## Projects

- **Basic HTML Website**: Core structure of a simple HTML website. [View on roadmap.sh](https://roadmap.sh/projects/basic-html-website)
- **GitHub User Activity**: Fetches and displays a user's public activity. [View on roadmap.sh](https://roadmap.sh/projects/github-user-activity)
- **Expense Tracker**: CLI to record and list expenses with budgets. [View on roadmap.sh](https://roadmap.sh/projects/expense-tracker)
- **Number Guessing Game**: CLI-based game, where you need to use the command line to interact with the game. [View on roadmap.sh](https://roadmap.sh/projects/number-guessing-game)

## Repository Structure

- `BackEndProjects/`: Backend and CLI projects.
- `FrontEndProjects/`: Web and UI projects.
- Additional directories will be added over time.

## Clone

```powershell
git clone <repo-url>
cd RoadMapsProjects
```

## Run Examples

Some projects include runnable scripts.

- Expense Tracker (from its folder):

```powershell
pnpm expense-tracker list
```
